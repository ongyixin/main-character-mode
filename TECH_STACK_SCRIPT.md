# Tech Stack Script (1 Minute)

**Main Character Mode** is built on a modern, AI-powered stack that brings real-time scene understanding to life.

**Frontend:** We're using **Next.js 16** with the App Router and Turbopack for lightning-fast development. The UI is built with **React 18**, styled with **Tailwind CSS 3**, and animated with **Framer Motion** for smooth, cinematic transitions.

**AI Engine:** The core intelligence comes from **Google's Gemini 2.0 Flash** — it processes camera frames in real-time to understand scenes, detect objects, and generate contextual narratives. For adaptive soundtracks that respond to gameplay, we integrated **Lyria 2** via Vertex AI. Visual assets — character cards, expressions, and recap posters — are generated by **NanoBanana** and Imagen 3.

**Vision Processing:** We use **MediaPipe Tasks Vision** for gesture detection, enabling natural interactions beyond just tapping.

**Export & Sharing:** Users can export their session recaps using **html-to-image**, **jsPDF**, and **Jimp** for image processing.

Everything is written in **TypeScript 5** for type safety and maintainability.

The architecture is a multimodal pipeline: camera frames flow to Gemini for scene understanding, which feeds our story engine, which orchestrates Lyria for music and NanoBanana for visuals — all synchronized in real-time to create a cohesive, adaptive experience.
